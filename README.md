# MasterPVZ_hack
Просто кряк лицензии для MasterPVZ

Актуально для v1.07.18

# Инструкция:
1. Скачать файл выше
2. Установить прогамму MasterPVZ
3. С рабочего стола правой кнопкой мыши по MasterPVZ -> Расположение файла
4. Закидываем файл который скачали ранее с заменой
5. Запускаем прогу и радуемся:)
